Ecommerce Website

This project was made using a pattern library, every pattern is coded in its own HTML file, in a pattern folder and draws from a CSS file in the pattern folder. To build each page, patterns are called out and pieced together will only minimal amounts of HTML to structure them. The site is completely responsive and accessible to people with disabilities.
